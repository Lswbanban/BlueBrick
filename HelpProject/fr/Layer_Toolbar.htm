The Layer Toolbar


The Layer Toolbar at the bottom of the layer pane provides the basic layer controls:

Moving layers

Moving layers to the front or back is done with buttons 1 and 2 respectively. It will move the contents of the layer tab 8, which you have selected.

Adding layers

Adding layers is done with buttons 3 through 6. Their functions will be discussed in the four different type of layers section. Note that each new layer you add, will be placed at the front. If you wish to have it below a previous layer you must use the "move layer down" button 2.

Removing layers

Removing layers can be done by selecting the layer tab and pressing button 7.

Layer properties

Layer properties can be changed by double clicking on the layers tab 8. This includes changing the layer name. Other properties differ depending on the layer type.

Layer visibility

Layer visibility can be changed by clicking on the eye icon 9 of the layer you want to hide, or unhide.
