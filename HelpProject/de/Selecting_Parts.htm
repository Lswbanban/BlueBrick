Auswahl von Teilen

Man kann ein einzelnes Teil (durch Anklicken) oder mehrere Teile (z.B. durch Ziehen der Maus über die auszuwählenden Teile) auswählen. Die Auswahl von Textfeldern erfolgt in der gleichen Weise wie die Auswahl von Teilen.

Die Auswahl mehrerer Teile erfolgt indem die STRG-Taste (Standardeinstellung) gedrückt wird und gleichzeitig die auszuwählenden Teile mit der linken Maustaste angeklickt werden. Wird bei gedrückter STRG-Taste ein bereits ausgewähltes Teil noch einmal angeklickt, so gehört dieses Teil danach nicht mehr zu den ausgewählten Teilen.

Eine Gruppe von Teilen wird ausgewählt, indem bei gedrückter linker Maustaste die Maus so bewegt wird, dass die auszuwählenden Teile innerhalb des rechteckigen Auswahlfeldes liegen. Durch Bewegung der Maus bei gleichzeitig gedrückter STRG-Taste (Standardeinstellung) und linker Maustaste können weitere Gruppen ausgewählt werden. Die so ausgewählten, einzelnen Gruppen werden zu einer Gesamtauswahl zusammengefasst. Wird die STRG-Taste nicht gedrückt, wird die vorherige Auswahl von Teilen rückgängig gemacht. Wird bei gleichzeitig gedrückter STRG-Taste und linker Maustaste das Auswahlfeld über bereits ausgewählte Teile gezogen, ohne dass neue Teile ausgewählt werden, so sind diese Teile danach nicht mehr Bestandteil der Auswahl.
